# NuclearCraft: Plutonium Age

## Phase 3 - Irradiated Zombie System

You are continuing development of NuclearCraft for PaperMC 1.21+ using Java 21.

Previous completed phases:

### Phase 1

* Core Framework
* Managers
* Commands
* GUI System
* Data System
* Config System

### Phase 2

* Radiation System
* Infection System
* Radiation Events
* Radiation Data
* Contagion Mechanics

Now implement the complete **Irradiated Zombie System**.

This system is the primary early-game progression path.

Players will encounter Irradiated Zombies before they discover Plutonium Ore.

The zombie system must be balanced, performant, configurable, and scalable.

---

# SYSTEM OVERVIEW

Irradiated Zombies are mutated zombies created by long-term plutonium contamination.

They are stronger than normal zombies.

They spread radiation.

They are the primary source of:

* Radioactive Core
* Mutated Seed
* Irradiated Heart

Every future progression system depends on players hunting these zombies.

# SPAWN DISTRIBUTION

This applies to:

* Surface Night Spawns
* Cave Spawns
* Structure Spawns
* Reinforcement Spawns
* Dungeon Spawns

Must NOT affect:

* Villager Zombie Conversions
* Custom Plugin Zombies
* Spawn Eggs
* Commands
* Mob Spawners (config option)

Create config settings for all sources.
