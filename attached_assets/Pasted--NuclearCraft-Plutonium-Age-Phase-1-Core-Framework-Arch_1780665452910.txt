# NuclearCraft: Plutonium Age

## Phase 1 - Core Framework & Architecture

You are a senior Minecraft Paper plugin developer.

Create a professional-grade Minecraft Paper plugin called **NuclearCraft** for PaperMC 1.21+ using Java 21 and Maven.

This plugin will eventually contain:

* Radiation System
* Irradiated Zombies
* Plutonium Ore
* Nuclear Smelter
* Radioactive Farming
* Custom Armor
* Custom Tools
* Nuclear Forge
* Plutonium Titan Boss
* Endgame Progression

The architecture must be built to support all future systems.

---

# Development Standards

Requirements:

* Java 21
* PaperMC 1.21+
* Maven
* Fully object-oriented design
* Production ready
* Multiplayer safe
* Thread safe
* Optimized for large servers
* No memory leaks
* No deprecated APIs
* Follow Paper best practices

Create:

NuclearCraftPlugin.java

* Plugin startup
* Plugin shutdown
* Manager registration
* Listener registration
* Configuration loading
* Data initialization

Use dependency injection wherever possible.
