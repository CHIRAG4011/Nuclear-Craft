# NuclearCraft: Plutonium Age

## Phase 2 - Advanced Radiation & Infection System

You are continuing development of the NuclearCraft Paper 1.21+ plugin.

Phase 1 has already been completed and includes:

* Core Framework
* Data System
* Config System
* Manager System
* Custom Item Framework
* Custom Block Framework
* Command Framework
* GUI Framework

You must now implement the complete Radiation & Infection System.

This system is the core mechanic of the entire plugin.

Everything else in NuclearCraft will eventually interact with this system.

The implementation must be production-ready, multiplayer-safe, highly optimized, configurable, and expandable.

---

# SYSTEM OVERVIEW

Radiation is a persistent disease-like status effect.

Unlike vanilla potion effects, radiation must:

* Persist through relogs.
* Persist through server restarts.
* Progress over time.
* Spread to other players.
* Be partially blocked by armor.
* Be removable through cures.
* Interact with future NuclearCraft systems.

Every player should have their own radiation profile.
